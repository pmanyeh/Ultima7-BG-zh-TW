# Ultima VII Traditional Chinese Localization (創世紀七代繁體中文在地化計畫)

這是一個致力於將經典角色扮演遊戲《創世紀七代：黑門》(Ultima VII: The Black Gate) 與資料片《巨蛇島》(Serpent Isle) 完整繁體中文化的開源專案。本專案配合 Exult 引擎的 UCC (Usecode C) 工具，將遊戲腳本進行翻譯、重新編譯並整合。

## 專案結構

- `BlackGate/` - 《黑門》相關的翻譯腳本與 Python 工具腳本
- `SerpentIsle/` - 《巨蛇島》相關的翻譯腳本與 Python 工具腳本
- `Translation_Guide_會增補.md` - 翻譯名詞對照與指導原則
- `BG_usecode_es_workflow.md` - 黑門腳本編譯與掛載工作流程
- `SI_usecode_es_workflow.md` - 巨蛇島腳本編譯與掛載工作流程

## 工作流程與編譯說明

詳細的腳本解譯、翻譯規則與使用 UCC 編譯的方式，請參閱專案內的 Markdown 說明檔。
